QR Code Generator:
-----------------

The QR Code Generator is a simple and interactive web application built using HTML, CSS, and JavaScript. It allows users to generate QR codes instantly by entering text or URLs. The generated QR code updates dynamically, making it easy to share links, messages, or any information in a scannable format.

 Features:
 ---------

Generates QR codes instantly based on user input

Supports text and URLs

Simple and responsive user interface

Built with pure HTML, CSS, and JavaScript (no external frameworks)

Technologies Used:
------------------

HTML5 – Structure of the web page

CSS3 – Styling and layout

JavaScript (ES6) – Logic for QR code generation

How It Works:
-------------

Enter any text or URL in the input field.

Click the Generate button.

Instantly view your QR code on the screen.
